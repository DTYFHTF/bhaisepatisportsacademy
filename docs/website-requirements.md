# Bhaisepati Sports Academy — Website Requirements

## Pages

| Page              | Purpose                                           |
|---|---|
| Home              | Hero, facilities overview, CTA to enroll          |
| About             | Academy story, mission, coaches                   |
| Badminton Training| Badminton programs, skill levels, coaching         |
| Gym & Fitness     | Gym facilities, conditioning programs              |
| Facilities        | Courts, gym, sauna overview                        |
| Programs          | Structured training programs with schedules        |
| Coaches           | Coach profiles and credentials                     |
| Events            | Tournaments, competitions, academy events          |
| Gallery           | Action shots, facility photos                      |
| Student Enrollment| Online registration form                          |
| News & Updates    | Academy news, achievements, announcements          |
| Contact           | Location, phone, form, map                         |

## Key Features

- Online student enrollment system
- Program selection and schedule visibility
- Age group and skill level filtering
- Coach profiles
- Facility showcase
- Event and tournament listings
- Gallery with action photography
- Mobile-first responsive design
- WhatsApp contact integration

## Tech Stack

- **Frontend:** Nuxt 4 (Vue 3) + Tailwind CSS
- **Backend:** Laravel 12 + Filament 3
- **Database:** SQLite (dev) / PostgreSQL (prod)
- **Auth:** Laravel Sanctum
- **Payments:** Khalti, eSewa, COD
- **SMS:** Sparrow SMS

## Design Direction

- Energetic, dynamic, sporty aesthetic
- Visual inspiration from shuttlecock movement
- Diagonal layout elements inspired by shuttle trajectories
- Strong athletic photography
- Competitive, youth-focused feel
- The site should feel fast and alive, like badminton itself
